# Input

The `Input` library deals with user input.

## Methods

---

### `Input.KeyPressed(vKey)`
Checks if a key was just pressed this frame.

#### Parameters

`vKey`
: (number) Virtual key code. Complete list can be found at [Microsoft Docs](https://learn.microsoft.com/en-us/windows/win32/inputdev/virtual-key-codes)

#### Returns

`pressed`
: (bool) Whether the key was just pressed this frame.
